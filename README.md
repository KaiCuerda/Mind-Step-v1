# 🧠 Neuro-Focus-Manager

### 🚀 El Proyecto
Neuro-Focus-Manager es una aplicación de escritorio minimalista diseñada para ayudar a gestionar la **carga cognitiva** diaria. El objetivo es proporcionar una interfaz visual limpia que permita al usuario elegir tareas basadas en su nivel de energía actual, evitando el agotamiento sensorial.

### 🛠️ Características Técnicas
* **Lenguaje:** Python 3.x
* **Interfaz Gráfica:** Tkinter (Diseño Minimalista)
* **Arquitectura:** Lógica basada en eventos y funciones modulares.
* **Estética:** Paleta de colores pastel sobre modo oscuro (#1e1e1e) para reducir la fatiga visual.

### 🎯 ¿Por qué este proyecto?
Como desarrollador de 15 años con TEA, busco crear herramientas que aporten orden y claridad. Esta app soluciona la dificultad de decidir "qué hacer" cuando la energía mental es variable.
